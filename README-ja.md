# HTML Document Editor

![](_asset/screenshot.png)

静的 HTML ドキュメントをプレビュー上で直接ビジュアル編集できる VS Code 拡張機能。
仕様書・レポート・デザインモックアップなど、インライン style を使った単一ファイル HTML を対象としています。

> この拡張機能は HTML ドキュメント専用です。React / Vue / Svelte などのコンポーネントフレームワークや、Tailwind のような CSS クラス駆動のスタックには対応していません。

## 機能

- プレビュー上の要素をクリックして選択
- クリック前にホバーハイライトで選択対象をプレビュー
- 右パネル (インスペクタ) — タイポグラフィ・サイズ・ボックスモデル・位置プロパティを編集
- カラーピッカー + 16進数テキスト入力によるカラー編集
- プロパティラベルを左右ドラッグして数値をスクラブ変更 (Figma スタイル)
- `px` / `rem` / `%` などの単位バッジをクリックして単位を切り替え
- 8方向リサイズハンドルで要素サイズをビジュアル調整
- ドラッグ&ドロップによる並べ替え (同階層の前後移動 + コンテナ内への移動に対応)
- 左パネル — 見出しアウトライン (h1〜h6) によるドキュメントナビゲーション
- 追加パネル — 見出し・段落・div・Flex Row・画像・リストなどのスニペットを挿入
- 最小差分書き戻し: 変更した属性またはテキストの範囲だけをソース上で差し替えるため、コメント・整形・空白は崩れない
- キーボードショートカット: `Ctrl+S` 保存、`Ctrl+Z` 元に戻す、`Ctrl+Y` / `Ctrl+Shift+Z` やり直し

## 動作要件

- VS Code 1.74 以降
- Node.js (ソースからビルドする場合)

## セットアップ

依存パッケージをインストールしてビルド:

```
npm install
npm run build
```

VS Code でこのフォルダを開き、`F5` キーで Extension Development Host を起動します。

ファイルをこのエディタで開くには:

1. VS Code で `.html` ファイルを開く
2. エディタタイトルバーの「ファイルを開くアプリケーションの選択...」ボタンをクリック
3. "HTML Document Editor" を選択

開発中はウォッチモードが便利です:

```
npm run watch
```
